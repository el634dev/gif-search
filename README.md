<!-- Installation -->
### Installation

<!-- Run Locally -->
### Run Locally
Clone the project using

```bash
  git clone
```

Go to the project directory

```bash
  cd gif-search-main
```

Install npm

```bash
   npm install
   npm init
```

Start the server, can use nodemon but you must install it using  `npm install nodemon -g`

```bash
   nodemon
```
